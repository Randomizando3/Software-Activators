# Software-Activators

Repositorio reservado para organizar materiais e documentacao relacionados ao tema de ativacao e licenciamento de software.

## Status atual

No momento, o repositorio possui apenas a estrutura inicial e ainda nao publica uma implementacao funcional.

## Observacao importante

Qualquer uso futuro deste repositorio deve respeitar licencas, termos de uso e politicas dos fabricantes de software envolvidos.
